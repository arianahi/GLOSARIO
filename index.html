glosario informatico

Un glosario es un catálogo alfabetizado de las palabras y expresiones de uno o varios textos que son difíciles de comprender, junto con su significado o algún comentario.

En la informatica o en las diferentes carreras tanto como en centros de servicio nos brindan información y/o explicaciones acerca de ciertos equipos en el cual puede ser seguro que no sepamos el significado del concepto. A continuacion les brindamos un glosario con las palabras básicas para asi tener un poco mas de conocimiento y lograr asi entender mejor la informacion y/o explicaciones que nos dan.

Actualizar: Volver a cargar o mostrar el contenido de una página Web o una ventana.

Archivo: Documento generado con una aplicación que se almacena en una unidad

Barra de tareas: Forma rectangular gris que aparece nada más cargar Windows 98 en la parte inferior del Escritorio y la cual se divide en zona de control, el botón Inicio, alguna barra de herramientas y la zona de carga de carpetas y aplicaciones.

BIOS: Información básica grabada en un chip imborrable que hace que un ordenador pueda funcionar y arrancar.

Chipset: Conjunto de chips que complementan el rendimiento del procesador principal.

Controlador (driver): Pequeño programa que sirve para reconocer y controlar un dispositivo de hardware específico.

Descargar: Copiar archivos de un equipo a otro mediante un módem o una conexión de red.

Desfragmentación: Proceso que consiste en escribir en clúster o pistas contiguas en una unidad la información existente en éstos, ya que se han podido fragmentar

(estar desperdigadas en distintas áreas del soporte donde se guardó). La fragmentación ralentiza la velocidad del disco y hace sufrir a la aguja.

Ejecutable: Dícese del archivo que puede poner en marcha un programa.

Formatear: Acción de dar formato o estructura a un disco o disquete en forma de sectores, pistas y clúster. Esta acción borra la información que contuviese el disco.

Hardware: Partes duras de un ordenador o componentes de éste.

HTML (Lenguaje de Marcas de Hipertexto): Lenguaje utilizado para crear páginas Web.

Importar: Transferir o enviar ficheros a otro programa distinto del que los generó.

Java: Lenguaje de programación creado por Sun Microsystem para proporcionar más velocidad y facilidad de uso a Internet, es independiente de la plataforma utilizada y está disponible para cualquier navegador de la WWW que admita este lenguaje..

Link: Cada uno de los enlaces de un módulo con las librerías que utiliza. En Internet, conexión de un documento con otro mediante un clic sobre un texto marcado o un icono o imagen.

Memoria virtual: Archivo del disco duro (conocido como fichero de intercambio) que el sistema operativo utiliza como si de memoria RAM se tratase, aunque bastante más lenta.

Navegador: Programa utilizado para acceder a los documentos almacenados en Internet.

Ofimática: Dícese de la informática y la tecnología aplicada a la oficina.

Panel de control: Grupo de herramientas que se utilizan para cambiar la configuración de nuestro equipo.

RAM (Random Access Memory): Memoria de Acceso Aleatorio que mantiene vivos los datos hasta que se desconecta el ordenador.

Resetear: Apagar el ordenador "in extremis" cuando el sistema se queda colgado, también se le llama Reinicializar el ordenador (descargar el sistema operativo para volverlo a recargar).

ROM (Read Only Memory): Memoria de sólo lectura. Controla las rutinas fundamentales para el funcionamiento de un ordenador y que no pueden ser borradas por el usuario ni aunque se desconecte la máquina.

Sistema operativo: Programa primario que debe tener un ordenador para que las demás aplicaciones puedan funcionar.

Software: Partes blandas de un ordenador o soportes donde se almacenarán los datos generados con éste.

Tarjeta de red: Hardware que se inserta en un equipo para conectarlo a una red.

Unidad de disco: Hardware en el que se pueden almacenar datos

Virus: Programas informáticos diseñados con mala intención, ya que se convierten en parásitos capaces de infectar a otros para incluir una copia evolucionada de sí mismos.

Web: World Wide Web, Internet. Zona gráfica compuesta por millones de páginas Web y a la cual accedemos por medio de un navegador.

Zócalo: Hueco o lugar de la placa base donde se insertan en microprocesador, las memorias u otros chips.
